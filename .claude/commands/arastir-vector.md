# /arastir-vector - Vector RAG Araştırması

`$ARGUMENTS` kritik noktasını yalnızca Vektör DB hattında araştır.

Sıra:
1. `hukuk_ara()` ile sorgu çalıştır.
2. Gerekirse kategori filtresi uygula.
3. Benzerlik skoru yüksek sonuçları seç.
4. Doktrin, strateji ve kavramsal çerçeveyi kısa raporla.
5. Çıktıyı ana araştırma raporuna girecek şekilde yaz.
